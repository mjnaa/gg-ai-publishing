  • 생성형 AI 서비스를 이용하기 위해 질문을 시작할 때는 좌측에 있는 사이드바 상단의 새채팅을 눌러요.
    화면 가운데 “안녕하세요? 오늘 무엇을 도와드릴까요?”가 보이면 질문하고 싶은 내용을 그대로 적으면 됩니다.
    어려운 말은 필요 없어요. “출장 보고서 초안 만들어줘”처럼 평소 말투로 써도 잘 알아들어요.

    • 이렇게 쓰세요 : 새채팅 → 채팅창에 원하는 질문 입력 → AI 답변 결과 확인
    • 예시 프롬프트 : "경기도 카페 텀블러세척기 지원사업 기획서 작성해줘"

  • 채팅창 아래 **파일추가(+)**를 눌러 보고서나 자료를 올리고 명령어를 입력하면 첨부파일문서를 참고해서 답변해주어요.
    길고 복잡한 문서도 핵심만 빠르게 정리됩니다.

    • 이렇게 쓰세요 : 파일추가 → 문서 올리기 → “3줄 요약” 또는 “표로 정리” 요청
    • 예시 프롬프트 : "이 문서를 5개의 핵심 포인트로 요약해줘", "표로 부서별 예산 항목을 정리해줘"

  • 자주 쓰는 유용한 프롬프트는 **스마트도우미**에서 바로 꺼내 쓸 수 있어요.
    스마트도우미에는 “보도자료”, “표준문서작성”, “요약”, “번역” 등 업무 메뉴가 준비되어 있어 손이 빨라집니다.

    • 이렇게 쓰세요 : 채팅창 아래에 있는 스마트도우미 선택 → 상세 유형 선택 → 채팅창에 원하는 질문 추가입력
    • 유의사항 : 템플릿은 기본 구조를 잡아줄 뿐, 세부 질문은 채팅창에 직접 입력하셔야 합니다

  • 공모/협약/행사/성과 등 보도자료 유형을 선택하면 형식에 맞춘 초안을 뽑을 수 있어요.
    제목, 부제목, 본문까지 자동으로 구성됩니다.

    • 이렇게 쓰세요 : 스마트도우미 버튼 → 보도자료 상세 유형 선택 버튼→ 추가세부 정보 입력 → 생성

  • 경기도청 한글 표준서식에 맞춰 문서를 자동 구성합니다.
    목차, 항목명, 문장 흐름까지 기본틀이 잡혀 있어서 작성 시간이 크게 줄어요.

    • 이렇게 쓰세요 : 스마트도우미 → 표준문서작성 → 서식 상세 유형 선택 → 상세내용 입력
    • 유의사항 : 기관 고유 양식에 포함된 상세내용들이 누락되지 않도록 입력해주세요.

  • 문서를 간단 요약(1~2문장), 문단 요약(각 단락 핵심), 의도/주제 요약(메시지 중심) 중 필요한 방식으로 정리할 수 있어요.

    • 이렇게 쓰세요 : 파일 첨부 또는 텍스트 붙여넣기 → 원하는 요약 타입 선택
    • 유의사항 : 회의 보고용이면 간단 요약, 장표용이면 문단 요약이 편합니다.

  • 외국어 문서나 메일이 필요할 때 영어/중국어/베트남어 등으로 번역을 요청하세요.
    톤(격식/친절/간결)도 함께 지시하면 더 자연스럽습니다.

    • 이렇게 쓰세요 : 텍스트 붙여넣기 → "영어로 번역” 버튼 선택 → 톤 추가 요청 입력
    • 예시 프롬프트 : 파일 첨부 또는 텍스트 붙여넣기 → 영어로 번역 선택 → “격식 있게" 등 톤 입력

    • 이렇게 쓰세요 : 새폴더 → 폴더명 입력(예: ‘출장보고서’) → 관련 채팅을 폴더에 추가
    • 유의사항 : 폴더를 삭제하면 폴더와 그 안의 채팅이 영구 삭제되니 신중히!

  • 폴더 이름은 언제든 폴더 이름바꾸기를 통해 변경할 수 있어요.

    • 예시 : 폴더에 마우스오버시 노출되는 옵션버튼 (…) 선택→ 이름바꾸기/삭제

    채팅 이름바꾸기로 제목을 명확히 하고, 자주 참고하는 채팅은 상단고정하세요.

    • 이렇게 쓰세요 : 채팅명에 마우스오버시 노출되는 옵션버튼 (…) 선택 → 상단고정/채팅이름 바꾸기
    • 예시 : "채팅 이름바꾸기 → ’25.09 민관협력 보도자료’" 후 상단고정하기
    • 유의사항 : 채팅이름은 쉽게 이해할 수 있는 이름을 권장합니다.

  • 프롬프트 라이브러리는 사용자가 자주 활용하는 지시문을 저장해두고 필요할 때 바로 불러와 사용할 수 있도록 구성된 기능입니다. 이를 통해 업무 문서 작성, 보고서 요약, 분석 요청 등 반복적으로 수행되는 작업을 보다 간편하고 효율적으로 자동화할 수 있습니다.

    프롬프트 라이브러리는 여러 메뉴로 구성되어 있으며,
    MY 메뉴에서는 개인 프롬프트를,
    부서 메뉴에서는 동일 부서 구성원이 공유한 프롬프트를,
    실·국 메뉴에서는 상위 조직 단위까지 공유된 프롬프트를,
    전체 메뉴에서는 기관 전체에 공개된 프롬프트를 확인할 수 있습니다.

    • 이렇게 쓰세요
      1) 화면 오른쪽 하단의 ‘+ 만들기’ 버튼을 클릭해 새 프롬프트 생성
      2) 프롬프트 명 입력 : 프롬프트의 사용 목적이 명확히 드러나도록 작성
      3) 프롬프트 상세 입력 : 생성형 AI에게 수행할 작업을 자연어로 구체적으로 설명
      4) 제공 범위 선택 : MY / 부서 / 실·국 / 전체 중 선택
      5) 저장 버튼 클릭 → 선택한 범위에 즉시 공유
      6) 목록에서 프롬프트 선택 후 ‘적용’ 버튼을 누르면 내용이 채팅창에 자동 삽입되며 필요 시 수정 후 바로 실행 가능
      7) 이미 저장된 프롬프트는 ‘보기 → 수정’ 메뉴에서 이름·내용·제공 범위 변경 가능하며, 삭제 버튼으로 손쉽게 제거 가능
    • 예시
      - 프롬프트 명 : “일반인도 이해하는 정책요약”
      - 프롬프트 상세 :
      “제공된 정책 문서를 기반으로 목적, 추진 배경, 주요 내용, 대상, 예산, 일정, 수행체계를 핵심만 간결하게 정리해 주세요.
      일반 시민도 이해할 수 있도록 전문 용어는 쉬운 표현으로 바꿔 설명해 주세요.
      정책이 해결하고자 하는 문제와 기대되는 변화를 명확히 보여주기 위해 정책 필요성을 2~3줄로 정리해 주세요.
      문서의 전체 내용을 5~7개 항목의 불릿 포맷 요약으로 구조화해 주세요.
      마지막에는 정책 시행을 통해 기대되는 효과를 1~2줄로 요약해 주세요.
      전체 문장 어조는 ‘공식 보고서용’에 맞게 간결하고 명확하게 작성해 주세요.”
    • 유의사항
      프롬프트 명과 상세 설명은 구체적일수록 안정적인 결과를 얻을 수 있어요.

  • 채팅 중 폰트사이즈 조절(작게/보통/크게/더크게)이 가능해요. 눈이 편해야 일이 빨라집니다.

    • 이렇게 쓰세요 : 채팅중 우측 상단의 글자 크기 버튼 사용 (또는 사이드바의 좌하단 계정 > 설정버튼을 눌러 폰트 사이즈를 조절할 수도 있어요)

  • 진행 중인 채팅도 나중에 찾기 쉽도록 폴더에 보관할 수 있어요. 불필요해지면 삭제로 정리합니다.

  • 생성된 내용은 표 복사, 다운로드 등을 통해 재활용할 수 있어요. 보고서 작성이 훨씬 쉬워집니다.

    • 이렇게 쓰세요 : 답변 영역의 표위에 마우스를 오버하면 표복사 버튼이 보입니다. 또는 생성된 답변의 하단에 보면 다운로드 버튼이 있어요. 다운로드 하기전에 미리보기 뷰어를 통해 내용을 확인할 수 도 있어요.

  • 답이 마음에 들면 좋아요, 부족하면 별로예요와 함께 이유를 남겨주세요.(스타일/지시미준수/사실아님/기타)

    • 이렇게 쓰세요 : 개인정보를 빼고 맥락만 적기
    • 예시 : "민원인의 연락처" 대신 "민원인 A"
    • 유의사항 : 개인정보가 들어간 파일도 업로드하지 마세요.

  • “채팅 기록은 시간이 지나거나 저장 공간이 부족해지면 오래된 것부터 자동 삭제”될 수 있어요.

    • 유의사항 : 이어서 대화는 어렵지만, 이전 채팅은 사이드바에서 다시 열어볼 수 있어요.

  • 좌측 사이드바를 닫으면 미니사이드바로 바뀌어 화면을 넓게 쓸 수 있어요.

    • 이렇게 쓰세요 : 설정 → 테마/폰트 조정, 필요 시 ‘모두 삭제’

  • 채팅을 하다 다른 LLM으로 이동하거나 기존 채팅 내용을 선택하면 현재의 채팅 흐름은 종료돼요.
    사이드바에서 다시 열어볼 수는 있지만 이어 대화는 불가합니다.

    • 이렇게 쓰세요 : 채팅종료시 종료 안내문구가 나오는데 다시 보지 않기’ 체크 시 안내창이 다시 나타나지 않습니다.

  • AI답변 생성후 채팅창 하단에 “AI 업무지원은 참고용” 안내가 있어요.
    중요한 내용은 반드시 담당자가 최종 확인하세요

    • 예시 : 중요 내용은 최신 본문 또는 내부 기준과 반드시 대조
    • 유의사항 : AI 답변만으로 결재하지 마세요.
